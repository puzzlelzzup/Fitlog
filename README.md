# **🏋️ Overview**

> _캘린더를 이용한 효과적인 다이어트 관리 앱 서비스_

다이어트에 성공하기 위해서는 오랜 기간 몸에 베인 습관을 바꿔야 합니다. 올바른 습관을 들이기 위해서는 생각보다 체계적인 방식으로 습관을 형성하고 지속해야 하죠. 동기가 불명확하거나, 투자 기간에 비해 적절한 목표를 세우지 못하거나, 꾸준한 동기부여가 부족한 경우 등에는 올바른 습관 형성을 위해 체계적인 접근 방식에 대한 가이드가 필요합니다. 이러한 유저에게 습관 형성을 위한 가이드라인이 되어줄 핏로그를 계획하게 되었습니다. 캘린더와 공유 기능을 제공하는 핏로그와 함께 나만의 운동 루틴을 기록하고, 건강한 다이어트 습관을 만들어요! 🔥

### 서비스 차별점
- 기존 앱: 일반적인 다이어트 앱은 "기록" 중심의 기능을 제공하는 경우가 많습니다.
- 차별점: 사진, 몸무게, 운동루틴, 식단을 하나의 카드로 통합하여 관리할 수 있습니다. 공유 기능을 통해 다른 사람과의 상호작용으로 지속적인 동기 부여를 얻을 수 있습니다.

### 프로젝트 진행 계획

- 앞으로의 진행 계획:
  1) 일별 기록 기능 통합 및 사진, 이모티콘 등록 기능 추가
  2) 로그인 및 공유 캘린더 앱과 연결
  3) 등록한 데이터 바탕으로 시각화 정보 제공

-  Branch:
  - 코드 관리를 위해 activate branch로 기능을 개발하고 있습니다. 개발 현황은 activate branch에서 확인할 수 있습니다.
  - main : 기본 branch (최종 완성본)
  - activate : 주요 기능 개발 및 업데이트 진행 중

# **🏋️ 주요 기능**

- `🖐️ 로그인`
  - 유저 로그인 / 회원가입을 통해 서비스 홈화면 접속
  - 유저별 일별 습관 카드와 캘린더 데이터 저장
- `📒 일별 습관 카드`
  - 카드 UI로 일별 데이터 저장
  - 사진, 몸무게, 운동루틴, 식단 등록 및 관리
  - 카드 하단부 업로드 버튼으로 공유 기능 적용
- `🗓️ 월별 습관 캘린더`
  - 날짜 터치를 통해 일별 습관 카드 열람
  - 날짜별 이모티콘 등록으로 월별 감정기록 한눈에 확인
  - 일별 등록한 몸무게 기반 변화 그래프 제공
- `👥 습관 스토리 공유`
  - 일별 습관 카드를 기반으로 전체 / 친구 공유
  - 댓글과 좋아요 기능으로 유저 간 소통
